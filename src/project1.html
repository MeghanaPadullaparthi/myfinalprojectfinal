1.Password

Every day we see thousands of articles on how important it is to have a strong password. Passwords should be put in password managers with a strong password protecting the password manager. Never answer your security questions correctly as it will lead different people to access your accounts easily. Don’t overlook your computer and smartphone passwords, don’t use short pins, use a full password involving letters, numbers and symbols.

2.Two factor authentications:

A two-factor authentication is when you are logging into a site it will first ask for your password then it is going to send you a code and will ask you to verify it. Phone numbers have proven multiple times to be a great security risk so many websites have now been starting to transition to using Google Authenticator.

3.Web browser:

You can use any web browser but make sure you are using only trusted add-ons and extensions. Also make sure your cookies, history, cache are always cleared.
